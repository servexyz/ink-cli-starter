# CLI-Starter

> Meow-based CLI app starter

## Getting Started

**Install**

```bash
git clone git@github.com:servexyz/cli-starter myProjectName
```

**Execute CLI**

```bash
npm start
```

**Execute CLI without Babel**

```bash
npm run dev
```

**Test CLI**

```bash
npm run test
```
